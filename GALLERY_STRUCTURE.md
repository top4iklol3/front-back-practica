# Структура папок для галереи

## Общая структура

Галерея использует следующую структуру папок:

```
Storage/
  └── gallery/                    ← Базовая папка галереи (resourceKey = "gallery")
      ├── 2024/                   ← Год проведения
      │   ├── футбол/             ← Категория/Карточка
      │   │   ├── photo1.jpg
      │   │   ├── photo2.jpg
      │   │   ├── photo3.png
      │   │   └── document.pdf
      │   ├── хоккей/
      │   │   ├── match1.jpg
      │   │   ├── match2.jpg
      │   │   └── results.pdf
      │   └── теннис/
      │       ├── tournament1.jpg
      │       └── tournament2.jpg
      ├── 2023/
      │   ├── футбол/
      │   │   └── photo1.jpg
      │   └── баскетбол/
      │       └── game1.jpg
      └── 2022/
          └── футбол/
              └── photo1.jpg
```

## Правила организации

### 1. Базовая папка
- **Название**: `gallery` (обязательно)
- **Расположение**: `Storage/gallery/`
- Это resourceKey для галереи

### 2. Папки годов
- **Название**: числовое значение года (например, `2024`, `2023`, `2022`)
- **Расположение**: `Storage/gallery/{год}/`
- Примеры: `2024`, `2023`, `2022`

### 3. Папки категорий (карточки)
- **Название**: название категории спорта (например, `футбол`, `хоккей`, `теннис`)
- **Расположение**: `Storage/gallery/{год}/{категория}/`
- Это и есть **карточки**, которые отображаются в галерее
- Каждая папка-категория = одна карточка

### 4. Файлы внутри категорий
- **Форматы изображений**: `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp`, `.svg`
- **Форматы документов**: `.pdf`
- Все файлы в папке категории будут отображаться как фотографии карточки

## Примеры правильной структуры

### Пример 1: Простая структура
```
Storage/gallery/
  2024/
    футбол/
      match1.jpg
      match2.jpg
```

**Результат**: 
- Карточка "футбол" за 2024 год
- 2 фотографии в карточке

### Пример 2: Несколько категорий
```
Storage/gallery/
  2024/
    футбол/
      photo1.jpg
      photo2.jpg
    хоккей/
      game1.jpg
      game2.jpg
      results.pdf
    теннис/
      tournament.jpg
```

**Результат**: 
- 3 карточки за 2024 год: "футбол", "хоккей", "теннис"
- В карточке "хоккей" будет 2 фото + 1 PDF

### Пример 3: Несколько лет
```
Storage/gallery/
  2024/
    футбол/
      photo1.jpg
  2023/
    футбол/
      photo1.jpg
    хоккей/
      photo1.jpg
```

**Результат**: 
- 2024 год: 1 карточка "футбол"
- 2023 год: 2 карточки "футбол" и "хоккей"

## Как создать структуру

### Через интерфейс хранилища:
1. Перейдите в раздел "Хранилище"
2. Создайте папку `gallery` (если её нет)
3. Внутри `gallery` создайте папку с годом (например, `2024`)
4. Внутри года создайте папку с категорией (например, `футбол`)
5. Загрузите фотографии в папку категории

### Через файловую систему:
1. Перейдите в папку `Storage/`
2. Создайте структуру папок: `gallery/2024/футбол/`
3. Скопируйте фотографии в папку `футбол/`

## Важные замечания

1. **Название папки года** должно быть числом (например, `2024`, а не `год-2024`)
2. **Название категории** - это название карточки, которое будет отображаться в галерее
3. **Фотографии** должны быть непосредственно в папке категории, не в подпапках
4. **Корзина** создается автоматически в папке `Storage/gallery/.trash/` при перемещении файлов

## Что отображается в галерее

- **Карточки**: каждая папка-категория = одна карточка
- **Название карточки**: название папки категории
- **Год**: название папки года
- **Категория**: название папки категории (то же, что и название карточки)
- **Количество фото**: количество файлов (изображений + PDF) в папке категории
- **Превью**: первое изображение из папки категории
